# Branch Protection Investigation Test Repository

This repository is used to investigate GitHub branch protection behavior for the Logik release branch locking utility.

## Purpose

Testing:
- Org-level rulesets vs repo-level branch protection
- Synchronization behavior of protection rules
- Override capabilities
- Permission requirements

## Test Branches

- `main` - Default branch
- `test-protection` - Basic protection testing
- `service/CY25/10.31` - Simulates a real release branch
- `service/CY25/11.15` - Another release branch for comparison
